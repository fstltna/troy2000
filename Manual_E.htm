OVERVIEW
TROY 2000 is a platform arcade action game. Your goal is to collect all the gold nuggets on each level and find a ladder to proceed to the next level in the tower. The tower has 55 levels and you will encounter 3 big bosses on the way to the top floor.

You can infinitely continue play on the same level. Also, it has automatic save feature so that you can start playing on the level you played last time.

This game is Shareware. Unregistered version is limited to the first 20 levels of play.

SYSTEM REQUIREMENTS
  • Windows 95/98/NT/2000 (Requires DirectX3.0 or higher)
  • Pentium 166MHz or faster
  • 16MB RAM
  • 5MB of free hard disk space
  • DirectDraw compatible Video Card
  • DirectSound compatible Sound Card
  • Joystick / Joypad recommended
KEY CONTROLS
  • Arrow Keys - Move Troy
  • [Shift] Key - Fire
  • [Ctrl] Key - Jump
  • [Esc] Key - Back to Menu
  • [P] Key - Pause

This game supports Joystick / Joypad.
